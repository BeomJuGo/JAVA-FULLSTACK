# Healthcare Platform

건강한 라이프스타일을 위한 헬스케어 플랫폼입니다.

## 프로젝트 구조

- **Backend**: Spring Boot (Java)
- **Frontend**: React + Vite
- **Database**: MySQL

## 기술 스택

### Backend
- Spring Boot
- Spring Security (JWT)
- Spring Data JPA
- Flyway (Database Migration)
- Cloudinary (Image Upload)

### Frontend
- React
- Vite
- Tailwind CSS
- Shadcn UI
- GSAP (Animations)
- FullCalendar

## 주요 기능

- 사용자 인증 및 권한 관리
- 트레이너 매칭 시스템
- 운동 및 식단 플랜 관리
- 커뮤니티 (게시글, 댓글, 좋아요)
- AI 트레이너 추천
- 실시간 채팅
- 리뷰 시스템
- 관리자 페이지

## 시작하기

### Backend 실행
```bash
./gradlew bootRun
```

### Frontend 실행
```bash
cd frontend
npm install
npm run dev
```
